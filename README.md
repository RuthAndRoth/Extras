# Extras

RuthAndRoth Extras is a collection of things related to Ruth2 and Roth2 like
optional accessories, clothing, logo textures, etc that were formerly
located directly in the Ruth repository, including:

* The ever-popular ElfEars are included in `Accessories/ElfEars`.
* Sundance Haiku's Fingernail Project and Hand Pose Library HUDs are included
in `HUDs`.
* The source Gimp XCF file for the R2 logo and a handful of pre-exported PNG
images are located in `Textures/Logo`.
* Some image files from Ada Radius are included in `Textures/Skin and Eyelash Helpers`.
* A number of images uploaded to the wiki used in illustrating mesh import
settings are in `Textures/Wiki`.
